Escreva menos CSS / Write less Css

Além das classes base, existem algumas outras como "padding-5, padding-20, margin-y-10..." que servem como apoio para rápida e facil moldagem de estilos dos seus elementos. Utilize estas classes de apoio sempre no final dos atributos "class" para a regra ser aplicada perfeitamente. Veja um exemplo:

<div class="base padding-20">Conteudo</div>
